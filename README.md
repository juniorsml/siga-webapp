# siga

Install yarn:

`https://yarnpkg.com/pt-BR/docs/install`

Install Packages

```
位 yarn
```

Run project

```
位 yarn start
```
The application is running at `localhost:4200`

Test project

```
位 yarn test
```

Build project

```
位 yarn build
```
Build Docker Container

```
位 docker build -t siga-app:1.0.0  . 
```
COWABUNGA! 

Run Docker Container

```
位 docker run -p 3000:80 siga-app:1.0.0
```

The application is running at `localhost:3000`
